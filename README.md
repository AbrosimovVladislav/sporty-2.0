# Sporty 2.0

Telegram Mini App для управления любительскими спортивными командами. Организация матчей, составов, статистики — всё внутри Telegram.

## Эпики

- [Онбординг](docs/features/onboarding/onboarding.md) — вход через Telegram, заполнение профиля
- [Команда](docs/features/team/team.md) — профиль команды, состав, заявки, права
- [Событие](docs/features/event/event.md) — создание, голосование, отметки посещения, финансы
- [Игрок](docs/features/player/player.md) — навигация, главная, профиль, история событий
