# Sporty 2.0

Telegram Mini App для управления любительскими спортивными командами. Организация матчей, составов, статистики — всё внутри Telegram.

## Эпики

- [Онбординг](docs/onboarding.md) — вход через Telegram, выбор роли, создание профиля игрока или команды
- [Команда](docs/team.md) — профиль команды, управление составом, заявки на вступление
- [Событие](docs/event.md) — создание событий (игра/тренировка/сбор), голосование, учёт посещений и финансов
- [Игрок](docs/player.md) — профиль игрока, лента событий, поиск команд
