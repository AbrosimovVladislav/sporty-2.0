# Онбординг

## Вход в приложение

Авторизация через Telegram — автоматически по initData при открытии Mini App. Отдельной формы логина нет.

При первом входе пользователь попадает на экран выбора роли.

## Экран выбора роли

Пользователь выбирает, зачем он пришёл:
- **Я игрок** → онбординг игрока
- **Создать команду** → онбординг организатора

Один пользователь может быть и игроком, и организатором (роль = контекст).

---

## Онбординг игрока

### Экран 1: Основная информация
- Имя (подтягивается из Telegram, можно изменить)
- Город
- Вид спорта (на старте только футбол)

### Экран 2: Готово
- Краткое объяснение: как искать команды и вступать
- Переход на главную игрока

### Результат
Создаётся профиль игрока. Пользователь попадает на главную — ленту событий / поиск команд.

---

## Онбординг организатора (создание команды)

### Экран 1: Информация о команде
- Название команды
- Вид спорта (на старте только футбол)
- Город

### Экран 2: Готово
- Краткое объяснение: как добавлять игроков, создавать события
- Переход на главную команды

### Результат
Создаётся команда. Пользователь становится организатором этой команды и попадает на главную команды.
