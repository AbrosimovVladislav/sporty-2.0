# Онбординг

## Вход в приложение

Авторизация через Telegram — автоматически по `initData` при открытии Mini App. Отдельной формы логина нет.

Во время загрузки Mini App показывается спиннер. Параллельно:
1. Приложение дергает `POST /api/auth/telegram` с `initData`.
2. Бэкенд валидирует подпись, апсертит пользователя в таблицу `User`.
3. Возвращает запись `User`.

После этого клиент решает, куда отправлять пользователя:
- `onboarding_completed === false` → экран онбординга
- `onboarding_completed === true` → главный экран приложения (`/home`)

## Онбординг — единый для всех

Без выбора роли. Ролей «игрок» и «организатор» на уровне пользователя не существует — роль живёт в `TeamMembership` и определяется действиями пользователя внутри конкретной команды.

### Экран: Профиль

Единственный экран онбординга. Пользователь заполняет:

- **Имя** — подтягивается из Telegram, можно поменять
- **Город** — свободный ввод
- **Вид спорта** — на старте только футбол, поле не редактируемое

Кнопка **«Продолжить»** вызывает `POST /api/onboarding` (серверная ручка через service-role клиент, чтобы не упереться в RLS). API обновляет `User`:
- `name`, `city`, `sport`
- `onboarding_completed = true`

После успешного ответа — редирект на `/home`.

### Результат

Создан полноценный профиль пользователя. Он попадает на главную с empty state: лента пустая, внизу блок «Ты пока не в команде» с кнопкой «Найти команду».

## Как стать организатором

Организатор — это не роль на уровне пользователя, а роль внутри команды (`TeamMembership.role`). Есть два пути получить её:

1. **Создать команду.** Пользователь нажимает «Создать команду» на табе «Команды», заполняет форму (название, город, спорт). При создании записи в `Team` автоматически создаётся `TeamMembership` с `role = 'organizer'`. Создатель = первый организатор команды.

2. **Получить промоут от существующего организатора.** Пользователь уже состоит в команде как игрок. Существующий организатор команды меняет его `TeamMembership.role` с `player` на `organizer`.

В одной команде может быть **несколько организаторов** — все с одинаковыми правами.

Вариант «подать заявку стать организатором извне» (не будучи игроком команды) — отложен за MVP, см. [post-mvp.md](post-mvp.md).
