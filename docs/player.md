# Игрок

## Профиль игрока

Карточка с информацией об игроке.

**Поля:**
- Имя
- Город
- Вид спорта
- Статистика посещений (кол-во событий, % явки)
- В перспективе: игровые характеристики (не в MVP)

---

## Главная игрока

### Лента событий
Список предстоящих событий из всех команд игрока:
- Карточка события: тип, дата, команда, площадка
- Быстрое голосование (приду / не приду) прямо из ленты

### Мои команды
Список команд, в которых состоит игрок:
- Название, вид спорта, кол-во игроков
- Переход в профиль команды

### История событий
Список прошедших событий:
- Был / не был
- Сдал / не сдал

---

## Поиск команд

Игрок ищет команды для вступления.

**Фильтры:**
- Город
- Вид спорта (на старте — только футбол)

**Результат поиска:**
- Карточка команды: название, город, кол-во игроков
- Кнопка «Подать заявку»

---

## Мои заявки

Список отправленных заявок на вступление в команды:
- Команда, дата заявки, статус (на рассмотрении / принята / отклонена)
