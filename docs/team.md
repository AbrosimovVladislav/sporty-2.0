# Команда

## Главная команды

Основной экран для организатора и игроков команды.

**Содержимое:**
- Название команды, вид спорта, город
- Кол-во игроков в составе
- Ближайшие события (список карточек)
- Общая статистика: кол-во проведённых событий, финансовый баланс

---

## Профиль команды

Публичная информация о команде (видна при поиске и заявке на вступление).

**Поля:**
- Название
- Вид спорта
- Город
- Количество игроков
- Описание (опционально)

---

## Управление составом

Доступно только организатору.

### Экран: Список игроков
- Список всех игроков команды (имя, дата вступления)
- Возможность удалить игрока из команды

### Экран: Заявки на вступление
- Список входящих заявок от игроков
- По каждой заявке: имя игрока, дата заявки
- Действия: принять / отклонить

---

## Поток заявки на вступление

1. Игрок находит команду через поиск
2. Игрок отправляет заявку
3. Организатор видит заявку в списке
4. Организатор принимает или отклоняет
5. Игрок получает результат (состоит / отклонён)
