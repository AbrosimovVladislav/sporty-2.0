# Заявки на вступление

Механизм присоединения к команде. Игрок подаёт заявку, организатор принимает или отклоняет.

## Флоу (со стороны игрока)

1. Игрок находит команду в каталоге (`/teams` → «Все команды»)
2. Открывает публичный профиль команды → [[team]-public-view.md]([team]-public-view.md)
3. Нажимает «Подать заявку на вступление» → создаётся `JoinRequest` со статусом `pending`
4. В профиле (`/profile` → «Мои заявки») игрок видит статус → [профиль](../player/[player]-profile.md)

## Обработка (со стороны организатора)

Точка входа: блок «Входящие заявки (N)» на главной команды → [[team]-home-tab.md]([team]-home-tab.md)

Карточка заявки: имя игрока, город, дата заявки.

Действия:
- **Принять** → `JoinRequest.status = 'accepted'`, создаётся `TeamMembership` с `role = 'player'`. Команда появляется у игрока в «Мои команды»
- **Отклонить** → `JoinRequest.status = 'rejected'`, в заявках у игрока остаётся запись со статусом
