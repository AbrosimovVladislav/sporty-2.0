# Создание команды

Доступно любому пользователю. Точка входа: экран «Команды» (`/teams`), secondary-ссылка «Создать команду».

## Форма

- Название команды
- Город
- Вид спорта (на старте — только футбол)

## Результат

1. Создаётся запись в `Team`
2. Создаётся `TeamMembership` с `user_id = текущий пользователь`, `role = 'organizer'`
3. Пользователь попадает на главную команды (`/team/[id]`)
