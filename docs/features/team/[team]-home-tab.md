# Таб «Главная команды»

Дашборд команды. Первое, что видит пользователь после захода в команду.

## Видит любой участник

- Количество игроков в составе
- Ближайшее событие (карточка-ссылка на детальный экран)
- Статистика: проведено / запланировано событий (если есть хотя бы одно)

## Дополнительно видит организатор

- Блок **«Финансовый баланс»** — реальный баланс команды (зелёный/красный), долги игроков и остаток к оплате площадкам мини-строкой. Кликабелен, ведёт на вкладку «Финансы». Формула — в [[team]-finances-tab.md]([team]-finances-tab.md)
- Блок **«Входящие заявки (N)»** с раскрытием списка заявок (если N > 0) → подробнее в [[team]-join-requests.md]([team]-join-requests.md)
