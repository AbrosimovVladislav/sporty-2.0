# Таб «Финансы команды»

Сводка по деньгам команды. Видит только организатор. Считается в реальном времени по завершённым событиям и расчётам с площадками.

## Реальный баланс

```
Реальный баланс = Касса − Остаток к оплате площадкам − Переплаты игроков + Задолженность игроков
```

Сколько денег **реально будет** у команды, когда все долги игроков получены и все обязательства площадкам оплачены.

Зелёный — профицит, красный — дефицит.

## Показатели

Считаются только по событиям с `status = 'completed'` (кроме площадочных полей — там учитываются все события с `venue_cost > 0`).

| Показатель | Формула |
|---|---|
| **Собрано от игроков** | Σ `getPaidAmount(a, price)` для attendances, где `paid_confirmed ?? paid = true` |
| **Ожидаемый сбор** | Σ `price_per_player` для attendances, где `attended_confirmed ?? attended = true` |
| **Долги игроков** | по каждому игроку: max(0, expected − paid); сумма таких положительных дельт по команде |
| **Переплаты игроков** | по каждому игроку: max(0, paid − expected); сумма таких положительных дельт по команде |
| **Расходы площадкам** | Σ `events.venue_cost` |
| **Оплачено площадкам** | Σ `events.venue_paid` |
| **Остаток к оплате площадкам** | расходы − оплачено (не < 0) |
| **Касса (на руках)** | собрано от игроков − оплачено площадкам |

`getPaidAmount(a, price)`:
- если `paid_confirmed ?? paid = true` и `paid_amount` указан — возвращает `paid_amount`
- если `paid_confirmed ?? paid = true` и `paid_amount = null` — возвращает `price`
- иначе 0

## Блоки страницы

1. **Реальный баланс** — крупная сумма, цвет зависит от знака
2. **Показатели** — карточки: Касса, Собрано, Расходы (план/факт), Долги, Переплаты
3. **Должники / переплатили** — списки игроков с ненулевыми дельтами; клик открывает [карточку игрока](../player/[player]-player-card.md)
4. **Площадки** — список событий с `venue_cost > 0`: тип, дата, площадка, стоимость и оплачено; клик ведёт на детальный экран события

## API

- `GET /api/teams/[id]/finances?userId=...` — все показатели + массивы `debtors[]`, `creditors[]`, `venueEvents[]`. Только организатор.
