# Таб «Финансы команды»

Сводка по деньгам команды. Видит только организатор. Считается из таблицы `FinancialTransaction`.

## Реальный баланс

```
Реальный баланс = Касса − Остаток к оплате площадкам − Переплаты игроков + Задолженность игроков
```

## Показатели

| Показатель | Формула |
|---|---|
| **Собрано от игроков** | Σ `amount` из `FinancialTransaction` (все типы) |
| **Ожидаемый сбор** | Σ `price_per_player` для attendances, где `attended = true`, событие завершено |
| **Долги игроков** | по каждому: max(0, expected − paid); сумма |
| **Переплаты игроков** | по каждому: max(0, paid − expected); сумма |
| **Расходы площадкам** | Σ `events.venue_cost` |
| **Оплачено площадкам** | Σ `events.venue_paid` |
| **Остаток к оплате** | расходы − оплачено (не < 0) |
| **Касса (на руках)** | собрано − оплачено площадкам |

## Блоки страницы

1. **Реальный баланс** — `StatCard` с крупным числом (40px), зелёный если ≥0, красный если <0
2. **Мини-метрики** — grid-2: «Долги игроков» (danger если > 0) и «К оплате площадкам» (warning если > 0)
3. **Касса** — Card: одна строка «На руках: N ₸»
4. **Сборы** — Card: «Ожидаемый сбор» + «Собрано от игроков»
5. **Расходы площадкам** — Card: «Всего» + «Оплачено» + «Остаток»
6. **Задолженности** — Card-список с аватаром игрока и суммой (−N ₸ красным, +N ₸ зелёным); клик открывает карточку игрока
7. **Площадки** — Card-список событий с `venue_cost > 0`; статус справа — «оплачено» или «−N ₸»; клик → событие
8. **BottomActionBar** — кнопка «Внести депозит»; модалка: игрок + сумма + комментарий → `POST /api/teams/[id]/transactions` с `type = deposit`

## Карточка игрока

Показывает сальдо и историю **транзакций** (не attendances):
- `event_payment` — привязан к событию, отображается с датой и типом
- `deposit` — общий депозит, отображается с комментарием

## API

- `GET /api/teams/[id]/finances?userId=...` — метрики + debtors + creditors + venueEvents. Только организатор.
- `POST /api/teams/[id]/transactions` — создать транзакцию (deposit или event_payment). Только организатор.
- `GET /api/teams/[id]/transactions?userId=&player_id=&type=` — список транзакций.
- `GET /api/teams/[id]/members/[userId]/finances?userId=...` — история транзакций + сальдо конкретного игрока.
- `GET /api/teams/[id]/members/[userId]/balance?userId=...` — только баланс (expected/paid/balance + transactions).
