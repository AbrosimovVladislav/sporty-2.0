# Таб «События»

Список событий команды — предстоящих и прошедших.

## Видит любой участник

- Список карточек событий: тип, дата, площадка, количество проголосовавших «приду»
- Клик по карточке → детальный экран события (см. [event.md](../event/event.md))

## Дополнительно видит организатор

- Кнопка **«Создать событие»** сверху экрана → [создание события](../event/[event]-create.md)
