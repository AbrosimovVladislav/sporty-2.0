# Публичный вид команды

Ограниченный view для показа команды «снаружи» — пользователь попал из каталога (`/teams` → «Все команды»).

## Содержимое

- Название, город, спорт
- Описание (если задано)
- Количество игроков в составе
- Кнопка **«Подать заявку на вступление»** (если заявки ещё нет)
- Либо надпись «Заявка отправлена» / «Заявка отклонена» (если уже подавал)

## Навигация

Суб-табы (Главная / Состав / События) показываются и в этом режиме — навигация единая для всех. Разница в содержимом: не-участник видит кнопку заявки вместо дашборда участника.
