# Онбординг

Пользователь открывает Mini App, авторизуется, заполняет профиль и попадает на главный экран.

Ролей на уровне пользователя нет — роль появляется внутри команды (см. [team.md](../team/team.md)).

- [Экран онбординга]([onboarding]-profile-setup.md) — форма профиля (имя, город, спорт)
