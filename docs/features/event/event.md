# Событие

Событие — любая активность команды: игра, тренировка, сбор и т.д. Создаётся организатором, игроки голосуют и отмечают участие.

## Статусы

- `planned` — предстоящее событие, идёт голосование
- `completed` — событие прошло, доступны отметки посещения и финансы
- `cancelled` — событие отменено
