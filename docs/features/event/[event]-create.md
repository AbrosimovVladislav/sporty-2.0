# Создание события

Доступно только организатору команды. Точка входа: кнопка «Создать событие» на табе «События» → [[team]-events-tab.md](../team/[team]-events-tab.md).

## Форма

- Тип события (игра / тренировка / сбор / другое)
- Дата и время
- Город (chips, дефолт — `team.city`; влияет на список площадок и создание новой)
- Площадка — выбор из существующих в выбранном городе (`/api/venues?city=X`) или создание новой. При смене города сбрасывается выбор площадки и района
- Стоимость участия (цена с игрока)
- Минимальное кол-во игроков для проведения
- Описание (опционально)

## Результат

Событие появляется в ленте команды со статусом `planned`. Игроки команды видят его и могут голосовать.
