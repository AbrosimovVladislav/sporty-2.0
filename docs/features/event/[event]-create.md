# Создание события

Доступно только организатору команды. Точка входа: кнопка «Создать событие» на табе «События» → [[team]-events-tab.md](../team/[team]-events-tab.md).

## Форма

- Тип события (игра / тренировка / сбор / другое)
- Дата и время
- Цена с игрока, минимальное кол-во игроков
- Город (`<select>`, дефолт — `team.city`; влияет на список площадок). При смене города сбрасывается выбор площадки и района
- Площадка — выбор из существующих в выбранном городе (`/api/venues?city=X`) или создание новой
- Стоимость площадки (`venues.default_cost`):
  - При выборе существующей площадки авто-подставляется её стандартная цена; организатор может изменить для конкретного события (это не меняет цену в БД)
  - При создании новой площадки введённая цена сохраняется как `default_cost` — следующий раз подставится автоматически
- Описание (опционально)
- Публичное событие (toggle) — видно в поиске вне команды

## Результат

Событие появляется в ленте команды со статусом `planned`. Игроки команды видят его и могут голосовать.
