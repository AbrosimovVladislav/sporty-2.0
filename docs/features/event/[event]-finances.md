# Финансы события

Финансовая сводка по завершённому событию. Видит только организатор.

## Показатели

- **Были на событии** — количество подтверждённых участников
- **Ожидаемый сбор** — были × цена с игрока
- **Сдали деньги** — количество из подтверждённых, кто сдал
- **Фактический сбор** — сдали × цена с игрока
- **Разница** — фактический − ожидаемый (профицит зелёным, дефицит красным)

## Принцип

Сбора денег через приложение нет — только учёт. Деньги передаются лично. Поля `paid` / `paid_confirmed` в `EventAttendance` фиксируют факт.

## Предварительная оценка (planned)

Для запланированных событий организатор видит упрощённый блок:
- Ожидаемый сбор = проголосовавшие «приду» × цена
