# Отметки посещения

Доступны только для завершённых событий (`status = 'completed'`).

## Единый список участников

На детальном экране завершённого события отображается один список строк с тогглами «Был» и «Сдал».

**Организатор** — может переключать оба поля для любого участника.

**Игрок** — может переключать оба поля только в своей строке.

**Остальные** — видят read-only индикаторы для всех строк.

Если значение не выставлено (`null`) — кнопка показывает «Был?» / «Сдал?»

## API

`PATCH /api/teams/[id]/events/[eventId]/attendance`

Body:
- Игрок за себя: `{ userId, attended?, paid? }`
- Организатор за другого: `{ userId, targetUserId, attended?, paid? }`
