# Карточка события

Основной экран события (`/team/[id]/events/[eventId]`). Видят все участники команды.

## Содержимое

- Тип, дата, время, площадка
- Стоимость участия
- Статус (запланировано / завершено / отменено)
- Описание (если есть)
- Минимальное количество игроков

## Блоки в зависимости от статуса

### Planned

- Кнопки голосования «Приду / Не приду» → [[event]-voting.md]([event]-voting.md)
- Список проголосовавших (придут / не придут)
- Организатор: кнопки «Завершить» / «Отменить» → [[event]-lifecycle.md]([event]-lifecycle.md)
- Организатор: ожидаемый сбор (предварительный)

### Completed

- Блок «Ваша отметка» (был/сдал) → [[event]-attendance.md]([event]-attendance.md)
- Организатор: финансовая сводка → [[event]-finances.md]([event]-finances.md)
- Организатор: список участников с кнопками подтверждения → [[event]-attendance.md]([event]-attendance.md)
- Не-организатор: список проголосовавших
