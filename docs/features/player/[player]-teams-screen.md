# Экран «Команды» (`/teams`)

Экран с двумя под-секциями (переключаются pill-табами сверху).

## 1. Мои команды

Список команд, в которых пользователь состоит.

**Карточка команды:**
- Название, город, спорт
- Количество игроков в составе
- Бейдж роли: «организатор» или «игрок»

Клик по карточке → `/team/[id]`.

**Empty state:** «Ты пока не в команде» + кнопка «Посмотреть все команды» (переключает на соседнюю под-секцию).

Внизу экрана — secondary-ссылка **«Создать команду»**. Ведёт на форму создания → [[team]-create.md](../team/[team]-create.md).

## 2. Все команды (каталог)

Каталог команд для поиска и вступления.

**Фильтры сверху:**
- Город
- Вид спорта (на старте — только футбол, зафиксировано)

**Карточки команд:**
- Название, город, спорт
- Количество игроков

Клик по карточке → публичный профиль команды → [[team]-public-view.md](../team/[team]-public-view.md). Оттуда можно подать заявку.
