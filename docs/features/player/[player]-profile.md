# Профиль (`/profile`)

Личные данные пользователя и его активность.

## Содержимое

- Имя, город, вид спорта
- **События** — история событий со сводкой → [[player]-event-history.md]([player]-event-history.md)
- **Мои заявки** — список поданных заявок со статусами (`на рассмотрении` / `принята` / `отклонена`) → подробнее о заявках: [[team]-join-requests.md](../team/[team]-join-requests.md)

Профиль — «я как пользователь», без привязки к конкретной команде. Отдельного «профиля игрока» и «профиля организатора» нет.
