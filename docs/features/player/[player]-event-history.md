# История событий в профиле

Секция «События» на экране профиля (`/profile`).

## Сводка

Показывается, если есть хотя бы одно завершённое событие:
- Завершённых событий: N
- Посещено: M из N

## Список

Последние 10 событий пользователя (из всех команд). Каждая карточка:
- Тип события (игра / тренировка / сбор / другое)
- Название команды + дата
- Бейджи: `приду` / `не приду` / `был`

Клик по карточке → детальный экран события → [[event]-detail.md](../event/[event]-detail.md).

## Связь

Данные берутся из `EventAttendance` с джойном на `Event` и `Team`. Подробнее об отметках: [[event]-attendance.md](../event/[event]-attendance.md).
