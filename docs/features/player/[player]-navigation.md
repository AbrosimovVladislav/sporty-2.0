# Глобальная навигация

Снизу фиксируются три таба, **всегда видимые** на всех экранах (в том числе внутри экрана команды):

```
[ Главная ]    [ Команды ]    [ Профиль ]
```

| Таб | Роут | Что содержит |
|-----|------|--------------|
| Главная | `/home` | Лента ближайших событий из всех команд пользователя |
| Команды | `/teams` | Мои команды + каталог всех команд с поиском |
| Профиль | `/profile` | Личная информация и настройки |

Глобальные табы не исчезают при переходе в детальные экраны (экран команды, экран события). Профиль и навигация между основными разделами всегда под рукой.

## Переходы между экранами

```
/home → карточка события → /team/[id]/events/[eventId]
/home (empty) → «Найти команду» → /teams (секция «Все команды»)

/teams «Мои» → карточка команды → /team/[id]
/teams «Все» → карточка команды → /team/[id] (вид не-участника)
/teams «Мои» → «Создать команду» → форма создания → /team/[id]

/profile → «Мои заявки» → список заявок
```

Возврат — через системный back или через табы.
