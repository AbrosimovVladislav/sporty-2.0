# Карточка игрока (TeamPlayerSheet)

Bottom sheet с профилем игрока в контексте конкретной команды: надёжность, финансы, последние события и actions организатора.

## Где открывается

- Таб «Состав» (`/team/[id]/roster`) — клик по строке игрока
- Таб «Финансы» (`/team/[id]/finances`) — клик по строке должника или переплатившего (открывается с `initialSection="finances"`)
- Главная команда (`/team/[id]`) — клик по компактной строке участника

Один компонент `TeamPlayerSheet` (`src/components/team/TeamPlayerSheet.tsx`).

## Header

- **Аватар** 84×84 круглый, тонкая обводка `--gray-200`. Без фото — серая инициалка
- **Overlay-плашка рейтинга** 30×30 в правом-нижнем углу: `--text-primary` фон, белый Oswald 12px, рамка 2.5px белая
- **Имя** — Oswald 24px uppercase
- Под именем — `flex-wrap` ряд:
  - Если `member.role === "organizer"` — pill «Организатор» с зелёной обводкой `1.5px solid var(--green-500)`, иконкой короны и текстом 11px uppercase
  - Если есть `city` — `📍 city` 13px secondary
- **Ряд бейджей**: `LevelChip` (hex `LevelBadge` + pill `«{skill_level} N/5»`), затем `PositionBadge` для каждой позиции
- **Кнопка «Открыть профиль»** — `h-12 rounded-full`, чёрная (`--text-primary`), белый текст 15px. Ведёт на `/players/[id]`

## Секции (аккордеон)

Каждая секция — карточка `bg-bg-secondary rounded-xl`, тап по заголовку раскрывает.

### Надёжность · в этой команде

Peek (свёрнуто): «{пропуски/отмены} • без нарушений», справа `{N}%` (зелёный ≥90, чёрный ≥70, красный иначе).

Раскрыто:
- Большое число `{N}%` + лейбл («Стабильный»/«Надёжный»/«Средняя»/«Низкая»)
- 2 mini-stat: Неприходы / Отмены
- Список последних событий: точка-маркер + лейбл («Сыграл матч» / «Не пришёл» / «Отменил» / «Записался — Тренировка») + дата справа + chevron

API: `GET /api/teams/[id]/members/[userId]/reliability?userId=requesterId`

### Финансы · в этой команде

Видна, если просмотрщик — организатор или сам игрок.

Peek: «Сдал X из Y ₸» + сальдо `±N ₸` (зелёный — переплата, красный — долг).

Раскрыто:
- Eyebrow «Переплата»/«Долг» + Oswald 28px число
- 2 mini-stat: Должен сдать / Сдал
- История транзакций (до 8): label + сумма (зелёный — депозит). Клик ведёт на событие

API: `GET /api/teams/[id]/members/[userId]/finances?userId=requesterId`

### Игровая статистика

Заглушка «🏗 Скоро — голы, передачи, MVP».

## Actions

Видны только организаторам (если просмотрщик — не сам игрок).

- **«Сделать организатором»** — `PATCH /api/teams/[id]/members/[memberId]` (бледно-серая кнопка)
- **«Удалить из команды»** — `DELETE /api/teams/[id]/members/[memberId]?userId=` (красная light-кнопка)

Если просмотрщик смотрит свой профиль — видна **«Покинуть команду»**: `h-12 rounded-full`, light-red bg, иконка logout + текст. Тап → подтверждение в той же зоне → `POST /api/teams/[id]/leave`.

## UI

- Бэкдроп `rgba(0,0,0,0.4)`, тап закрывает
- Sheet: white bg, `rounded-t-[24px]`, max-h `88vh`, скролл внутри
- Drag-handle 36×4 + крестик (28×28, `--gray-100` bg) сверху
- Уровень/позиции и `LevelChip` импортируются из `@/components/players/badges`
